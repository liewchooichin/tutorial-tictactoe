Using Unicode characters in HTML

Enter the unicode character like this
&#x<UNICODE>;
Remember to end it with the ;

Examples:

  • 🗴    &#x1F5F4;      Ballot script X
  • 🗵    &#x1F5F5;      Ballot box with script X
  • 🗸    &#x1F5F8;      Light check mark
  • 🗹    &#x1F5F9;      Ballot box with bold check
  • 🌟    &#x1F31F;      Glowing star
  • 🌠    &#x1F320;      Shooting star
  • 🌡    &#x1F321;      Thermometer
  • 🌣    &#x1F323;      Sunny
  • 🌤    &#x1F324;      White sun with small cloud
  • 🌥    &#x1F325;      White sun behind cloud
  • 🌧    &#x1F327;      Cloud with rain
